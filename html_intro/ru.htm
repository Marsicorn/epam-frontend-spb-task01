HTML

HTML

HTML (HyperText Markup Language) – язык разметки гипертекста.

HTML не является языком программирования, он предназначен для структурирования документов без привязки к средставм отображения.

История HTML

  1. SGML (Standard Generalized Markup Language)
  2. HTML1 by Sir Timothy John "Tim" Berners-Lee
  3. HTML 2.0 (1995)
  4. HTML 3 (1996)
  5. HTML 4.0 (1997)
  6. HTML 4.01 (1999)
  7. HTML 5 (в разработке W3C и WHATWG с 2004)

Атрибуты

Атрибуты

Атрибут – пара вида свойство="значение", представляющие дополнительную информацию об элементе

  • глобальные/неглобальные
  • обязательные/необязательные

Элементы

Элементы

				<p>Это параграф</p>
				<a href="http://yandex.ru">Яндекс</a>
				<img src="pictures/logo.png" alt=""/>

Синтаксис элемента

  • Элемент HTML начинается с открывающего тега
  • Элемент HTML заканчивается закрывающим тегом
  • Содержимое элемента это все между началом и концом тега
  • Некоторые HTML элементы имеют пустое содержимое
  • Пустые элементы закрыты в начале тега

Уровни семантики

  1. Применение HTML-элементов
  2. Именование элементов

Первый уровень: применение HTML-элементов

				<div id="menu">
				  <a href="#">Колбаса</a>
				  <a href="#">Масло</a>
				</div>

Первый уровень: применение HTML-элементов

				<ul id="menu">
				  <li><a href="#">Колбаса</a></li>
				  <li><a href="#">Масло</a></li>
				</ul>

Второй уровень: именование

				<div id="header">
